# Hate Speech Detection using Machine Learning

## 📌 Overview

This project aims to detect and classify hate speech in textual data using Natural Language Processing (NLP) and Machine Learning techniques. The model analyzes social media text and predicts whether the content contains hate speech.

## 🚀 Features

* Text preprocessing and cleaning
* Stopword removal and text normalization
* TF-IDF vectorization
* Machine Learning-based classification
* Performance evaluation using standard metrics
* Custom text prediction

## 🛠️ Technologies Used

* Python
* Google Colab
* Pandas
* NumPy
* NLTK
* Scikit-learn
* Matplotlib

1. Data Collection

7. Prediction

## 📊 Dataset

The dataset contains labeled text samples categorized as hate speech, offensive language, or non-hate speech.

## 📈 Results

The trained model successfully classifies textual content into appropriate categories. Model performance was evaluated using accuracy, precision, recall, and F1-score metrics.
